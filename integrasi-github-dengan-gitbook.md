#### Integrasi GitHub dengan GitBook

```markdown
# Integrasi GitHub dengan GitBook

GitBook dapat dihubungkan dengan **GitHub** untuk otomatisasi dokumentasi.

## 1️⃣ Menyambungkan GitBook ke GitHub
1. Masuk ke **Settings** di GitBook.
2. Pilih **Integrations** → **GitHub**.
3. Hubungkan dengan akun GitHub.
4. Pilih repository untuk menyimpan dokumentasi.

## 2️⃣ Sinkronisasi Otomatis GitHub ke GitBook
Setiap perubahan di GitHub akan **otomatis diperbarui** di GitBook.

**Workflow**:
1. Buat perubahan di repository GitHub.
2. Commit dan push ke branch utama.
3. GitBook akan memperbarui dokumentasi.

## 3️⃣ Workflow Dokumentasi dengan Git
- **Buat branch baru** untuk perubahan.
- **Edit dokumentasi Markdown**.
- **Commit perubahan** dan lakukan **pull request**.
- **Merge ke branch utama**.
- **Sinkronisasi dengan GitBook**.
```
